Сетка

Подготовка
Параметры

Создает элемент «Сетка» из базовой линии с заданными параметрами: отдельная ширина слева и справа.

Порядок работы
  1. Выберите базовую линию.
  2. При необходимости укажите поверхность сетки для высот.
  3. Настройте параметры и запустите создание.

Если сетка не выбрана, она создается на Z = 0. Если выбрана поверхность, используется ее рельеф с учётом offset. Значение 0 допускается для любой стороны; отрицательные значения воспринимаются по модулю.
