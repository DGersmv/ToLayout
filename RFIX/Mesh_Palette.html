Сетка

Создает элемент «Сетка» из базовой линии с заданными параметрами.
Порядок работы:
1. Нажмите «Выбрать линию» для указания базовой линии.
2. (Опционально) Нажмите «Установить сетку», чтобы выбрать поверхность для высот.
3. Задайте параметры и нажмите «Создать сетку из линии».

Примечание: Если сетка не установлена, она будет создана на Z=0.0 (или Z=offset). Если сетка установлена, используется поверхность рельефа с учётом offset.
