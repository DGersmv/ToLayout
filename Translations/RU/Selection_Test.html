Create Contours from Line
Creates contour lines from base line with specified parameters.
For spline: places points with given step, builds perpendiculars
For other lines: checks closure, builds perpendiculars at start/end
Creates two side contours (left/right Splines) with closing lines (for open lines).
Attention: First select base line (spline/polyline/arc/line).

Workflow:
1. Click "Select Line" to select base line
2. Set Width and Step parameters
3. Click "Create Contours" - creates only contour lines (left/right Splines + closing lines)
Create Mesh from Line
Creates Mesh element from base line with specified parameters.
Workflow:
1. Click "Select Line" to select base line (spline/polyline/arc/line)
2. Set Width and Step parameters
3. (Optional) Click "Set mesh" to select mesh surface for Z-coordinates
4. Click "Create Mesh from Line" - creates Mesh from closed contour

Note: If mesh is not set, Mesh will be created at Z=0.0 (or Z=offset if specified). If mesh is set, Mesh follows the terrain surface with offset applied to Z coordinates.
