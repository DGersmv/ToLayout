Подсказка: установите угол или выберите действие
Привязка
Инструкции:
1. Выберите 3D сетку и объекты для привязки одновременно
2. Нажмите "Привязать к сетке" для размещения объектов на 3D сетке

Смещение: "Применить" — простое смещение по Z (без 3D сетки)
Размеры и разметка
Инструкции:
1. Создать разметку:
• Установите шаг разметки (например, 1000 мм)
• Нажмите "Установить" для применения шага
• Выберите объекты для разметки (Плита, Сетка, Оболочка, Стена) или линии (Линия, Полилиния, Дуга, Сплайн)
• Нажмите "Создать разметку"

2. Размеры к линии:
• Выберите объекты (Объект/Колонна/Лампа)
• Нажмите "Размеры к линии"
• Укажите две точки линии

3. Размеры между объектами:
• Выберите объекты (Объект/Колонна/Лампа)
• Нажмите "Размеры между объектами"

4. Размеры к точке:
• Выберите объекты (Объект/Колонна/Лампа)
• Нажмите "Размеры к точке"
• Укажите целевую точку
Создать контуры из линии
Создает контурные линии из базовой линии с заданными параметрами.
Для сплайна: размещает точки с заданным шагом, строит перпендикуляры
Для других линий: проверяет замкнутость, строит перпендикуляры в начале/конце
Создает два боковых контура (левый/правый Сплайны) с замыкающими линиями (для открытых линий).
Внимание: Сначала выберите базовую линию (сплайн/полилиния/дуга/линия).

Порядок работы:
1. Нажмите "Выбрать линию" для выбора базовой линии
2. Установите параметры Ширина и Шаг
3. Нажмите "Создать контуры" — создаются только контурные линии (левый/правый Сплайны + замыкающие линии)
Создать сетку из линии
Создает элемент Сетка из базовой линии с заданными параметрами.
Порядок работы:
1. Нажмите "Выбрать линию" для выбора базовой линии (сплайн/полилиния/дуга/линия)
2. Установите параметры Ширина и Шаг
3. (Опционально) Нажмите "Установить сетку" для выбора поверхности сетки для Z-координат
4. Нажмите "Создать сетку из линии" — создается Сетка из замкнутого контура

Примечание: Если сетка не установлена, Сетка будет создана на Z=0.0. Если сетка установлена, Сетка следует поверхности рельефа.
Массовая установка ID для выбранных элементов
Введите базовое имя и нажмите кнопку. Все выбранные элементы получат ID с номером (например: Сосна обыкновенная-01, Сосна обыкновенная-02...)
Создать папку/слой и переместить элементы

Разделяйте подпапки символом "/" (например: "Ландшафт/Растения/Деревья")
Что произойдет:
• Будет создана папка слоя (если она не существует)
• В этой папке будет создан новый слой
• Все выбранные элементы будут перемещены в этот слой
• Если установлена галочка "Скрыть созданный слой", слой будет скрыт
• ID элементов останутся прежними (могут быть изменены отдельно)
• Это можно отменить с помощью Ctrl+Z
Ориентация балок


Инструкции:
1. Выберите балки и нажмите "Установить балки"
2. Выберите поверхность сетки и нажмите "Установить сетку"
3. Нажмите "Ориентировать к поверхности" для выравнивания балок по сетке
Примечание: Длина балки сохраняется, направление балки следует касательной плоскости поверхности
Повернуть выбранное

Инструкции:
1. Выберите балки
2. Введите угол поворота в градусах (положительный или отрицательный)
3. Нажмите "Повернуть" для поворота выбранных балок
Примеры:
• +180° - повернуть на 180 градусов по часовой стрелке
• +90° - повернуть на 90 градусов по часовой стрелке
